# AI.FileOrganizer

AI.FileOrganizer is an AI-powered command-line tool for organizing and categorizing files and folders on your computer. It uses the [Microsoft Agent Framework](https://learn.microsoft.com/en-us/agent-framework/overview/) with support for multiple AI providers to interpret natural language instructions and automate file management tasks.
